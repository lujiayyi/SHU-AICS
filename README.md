# 上海大学智能计算系统

本仓库整理了上海大学智能计算系统课程的复习资料，包含课程 PPT、习题集、实验以及考试回忆等内容。

## 资料目录

### 课程 PPT

- **第一～九章PPT.pdf** - 课程主要内容（第1-9章）
- **大模型训练PPT.pdf** - 大模型训练专题
- **拓展PPT并行计算与异构计算、基于GPU的智算系统、大模型系统.pdf** - 并行计算、异构计算、GPU智算系统等拓展内容

### 实验资料

- **补充实验教材.pdf**  
  > 课程实验的补充指导材料

- **实验代码**  
  > 包含以下实验内容：

  - **v3.0 实验 3-1**：基于 **VGG19** 的图像分类  
  - **v3.0 实验 3-2**：基于 **DLP 平台** 的图像分类  
  - **v3.0 实验 3-3**：非实时图像风格迁移  
  - **v3.0 实验 8-1（opt）**：Stable Diffusion 实现及 **Triton 优化**  
  - **v3.0 实验 8-2（opt）**：Llama 3.2 实现及 **Triton 优化**  
  - **实验 8-3（opt）**：Code Llama 实现及 **Triton 优化**

### 习题
